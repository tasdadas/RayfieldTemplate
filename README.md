# About 📓
Please make sure to check out the official rayfield [links](#links-) to find the original documentation and to learn how to config this template.

I will be making a template for Rayfield Gen2 soon, however you can learn to use Gen2 by visiting the official Gen2 documentation (https://docs.sirius.menu/rayfield-gen2)

You don't really need this script to make a Rayfield GUI, building this template was extremely easy and done in about 10 minutes.
All this really does is just showcase the features of Rayfield and remove comments from the documentation snippets.
**All of the interactive elements simply just print to the console once used.**

If you have any issues please let me know I am somewhat active on here!
